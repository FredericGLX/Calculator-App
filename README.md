# Calculator App

- A basic calculator application including a dark theme.
- Front-end design from [BigsonDev](https://bigsondev.com/projects/calculator-app-project/).

### Concepts used

- ES6 object classes
- DOM Manipulation
- Ternary Operator
- CSS Grid
- Enabling Dark Mode
